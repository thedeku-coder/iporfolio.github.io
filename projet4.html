Base de Données pour les Jeux Olympiques

La conception d'une base de données SQL pour la gestion des données des Jeux Olympiques implique la création d'une structure organisée pour stocker et interagir avec les informations clés relatives à cet événement mondial. Cette base de données sera le pivot central pour gérer les détails sur les athlètes, les sports, les événements, les pays participants, les médailles, les records, ainsi que les informations administratives telles que les éditions spécifiques des Jeux Olympiques et le personnel impliqué.

❮ ❯

Competences

Ce projet visé a développé les compétences et savoir-faire informatique suivante :

• ventiller les données dans les différente table

• gestion des tables

• creation des tables
